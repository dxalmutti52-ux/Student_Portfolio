## 🚗 Car Insurance – ASP.NET MVC Application

A full MVC web application that mimics a real car insurance quote system.

### ✔ Features
- Takes user input (age, car year, car make/model, DUI, speeding tickets, coverage type)
- Calculates insurance quotes using business logic
- Stores quotes in a SQL Server database using Entity Framework
- Includes an Admin Dashboard to view all issued quotes

### ✔ Technologies
- ASP.NET MVC  
- Entity Framework 6  
- SQL Server  
- Razor Views  
- C#
